# iriscli stake pool

## 描述

查询最新的权益池

## 用法

```
iriscli stake pool <flags>
```

打印帮助信息
```
iriscli stake pool --help
```

## 示例

查询最新的权益池
```
iriscli stake pool
```

运行成功以后，返回的结果如下：

```txt
Pool
    Loose Tokens: 49.8289125612
    Bonded Tokens: 100.1800000000
    Token Supply: 150.0089125612
    Bonded Ratio: 0.6678269863
```
