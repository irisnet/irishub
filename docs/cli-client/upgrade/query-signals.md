# iriscli upgrade query-signals

## Description

Query the current information of signals.

## Usage

```
iriscli upgrade query-signals
```

Print help messages:

```
iriscli upgrade query-signals --help
```
## Flags

| Name, shorthand | Default                    | Description                                                       | Required |
| --------------- | -------------------------- | ----------------------------------------------------------------- | -------- |

Query the current information of signals.

iriscli upgrade query-signals
